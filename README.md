# Word Page Extractor

אפליקציה מקומית ל-Windows שמחלצת עמוד או טווח עמודים ממסמך Word (`.docx`) למסמך Word חדש, תוך ניסיון לשמור על עיצוב, כותרות עליונות/תחתונות ולוגואים.

> חשוב: Word אינו שומר “עמודים” כיחידות תוכן קבועות. העמודים נוצרים בזמן העימוד. לכן הכלי משתמש ב-Microsoft Word המותקן במחשב כדי לזהות את גבולות העמוד בפועל.

## מצב הפרויקט

גרסה ראשונית / Proof of Concept.

יכולות קיימות:

- בחירת קובץ DOCX.
- בחירת עמוד התחלה ועמוד סיום.
- יצירת מסמך DOCX חדש.
- העתקת התוכן בפורמט המקורי ככל האפשר.
- העתקת Header/Footer מה-Section שבו מתחיל הטווח.
- שמירת הגדרות עמוד בסיסיות: גודל דף, שוליים וכיוון.

מגבלות ידועות:

- נדרש Windows.
- נדרש Microsoft Word מותקן.
- טבלאות שחוצות עמודים עשויות להיחתך באופן לא מושלם.
- תמונות צפות או אובייקטים שמעוגנים לפסקאות מחוץ לעמוד עשויים לא לעבור נכון.
- Header/Footer מורכבים עם כמה Sections עשויים לדרוש שיפור נוסף.

## התקנה למפתחים

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## בניית קובץ EXE

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name WordPageExtractor app.py
```

הקובץ יווצר בתיקיית `dist`.

## שימוש

1. פותחים את האפליקציה.
2. בוחרים קובץ Word.
3. מזינים עמוד התחלה ועמוד סיום.
4. בוחרים היכן לשמור את הקובץ החדש.
5. לוחצים Extract.

## למה לא דפדפן בלבד?

דפדפן רגיל לא יודע לחשב את עימוד Word כמו Microsoft Word. לכן הכלי רץ מקומית ומשתמש ב-Word Automation דרך COM.
